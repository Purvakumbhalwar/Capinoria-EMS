# Capinoria EMS

This repository contains the Capinoria EMS project (frontend + backend).

Quick start:

1. Install dependencies (root and backend):

```powershell
npm install
cd backend
npm install
cd ..
```

2. Start the dev server (frontend):

```powershell
npm run dev
```

3. Start the backend server:

```powershell
cd backend
node server.js
```

Notes:
- Add a `.env` file in `backend/` if required by the backend server.
- To upload this project to GitHub, create a repo on GitHub and follow the push instructions in the project root.
